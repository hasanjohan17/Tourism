# Admin Dashboard Setup Guide

## كيفية إعداد لوحة التحكم الإدارية

### 1. تفعيل حساب مسؤول
أولاً، تحتاج إلى تحويل مستخدم إلى مسؤول:

#### الطريقة 1: من Supabase Dashboard
1. اذهب إلى Supabase Dashboard
2. اختر قاعدة البيانات الخاصة بك
3. اذهب إلى **SQL Editor**
4. قم بتشغيل الأمر التالي (استبدل user_email بالبريد الفعلي):

```sql
-- تحويل مستخدم إلى مسؤول
UPDATE public.profiles
SET role = 'admin'
WHERE id IN (
  SELECT id FROM auth.users 
  WHERE email = 'your-admin-email@example.com'
);
```

### 2. الوصول إلى لوحة التحكم
بعد تسجيل الدخول كمسؤول:
1. اضغط على زر **My Account** أو قم بتسجيل الدخول
2. ستجد زر جديد **Admin Dashboard**
3. اضغط عليه للوصول إلى لوحة التحكم

### 3. ميزات لوحة التحكم

#### أ) تبويب المستخدمين (Users Tab)
- عرض جميع المستخدمين المسجلين
- البحث عن المستخدمين حسب الاسم أو البريد الإلكتروني
- عرض المعلومات الشخصية:
  - الاسم الكامل
  - البريد الإلكتروني
  - رقم الهاتف
  - المنطقة/الدولة
  - حالة جواز السفر (تم الرفع أم لا)
  - تاريخ التسجيل

#### ب) تبويب المناشير (Destinations Tab)
- عرض جميع المناشير السياحية
- إضافة منشور جديد
  - العنوان (Title)
  - الوسم (Tag)
  - وصف (Description)
  - رابط الصورة (Image URL)
  - الموقع (Location)
  - ترتيب العرض (Sort Order)
  - حالة النشر (Published/Draft)
- حذف المناشير

#### ج) تبويب الخطط (Travel Plans Tab)
- عرض جميع خطط الرحلات
- إضافة خطة جديدة
  - الكود (A, B, C)
  - الاسم
  - الوصف الفرعي
  - المدة الزمنية
  - السعر
  - رابط الصورة
  - لون الاهتمام (Marine, Jade, Saffron)
  - وضع مميز (Featured)
  - حالة النشر (Published/Draft)
- حذف الخطط

### 4. قاعدة البيانات

الجداول المستخدمة:
- `profiles` - بيانات المستخدمين
- `destinations` - المناشير السياحية
- `travel_plans` - خطط الرحلات

### 5. ملاحظات مهمة

- **الأمان**: لوحة التحكم محمية وتحتاج إلى تسجيل دخول كمسؤول فقط
- **أسماء الصور**: استخدم النموذج `/images/filename.jpg` لأسماء الصور
- **الترتيب**: استخدم حقل Sort Order لتحديد ترتيب ظهور العناصر
- **الحالة**: اختر Published لجعل العنصر مرئياً للعملاء

### 6. مثال على إضافة منشور

1. اختر تبويب **Destinations**
2. اضغط على **Add Destination**
3. ملأ البيانات:
   - Title: "Damascus Old City"
   - Tag: "Living history"
   - Image URL: "/images/damascus umaayya.jpg"
   - Location: "Old Damascus"
4. اضغط **Add Destination**

### 7. استكشاف الأخطاء

**المشكلة**: لا أرى زر Admin Dashboard
- **الحل**: تأكد من أن حسابك مسؤول (تحقق من دور المستخدم في قاعدة البيانات)

**المشكلة**: فشل إضافة منشور
- **الحل**: تأكد من:
  - وجود اتصال بـ Supabase
  - صحة اسم ملف الصورة
  - ملء جميع الحقول المطلوبة

**المشكلة**: الصور لا تظهر
- **الحل**: تأكد من أن الصور موجودة في مجلد `public/images/`
